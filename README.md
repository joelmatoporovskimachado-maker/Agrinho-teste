# 🌱 Agro Forte — Futuro Sustentável

Projeto de página web front-end com foco no agronegócio sustentável, destacando o equilíbrio entre produção agrícola, tecnologia e preservação ambiental.

---

## 📖 Sobre o Projeto

O **Agro Forte — Futuro Sustentável** foi desenvolvido com o objetivo de apresentar uma interface moderna e responsiva voltada ao setor agropecuário sustentável.

A proposta visual busca transmitir:

- Sustentabilidade
- Inovação no campo
- Tecnologia agrícola
- Preservação ambiental
- Produção consciente

---

## 🚜 Tema

**“Equilíbrio entre produção e meio ambiente”**

O projeto enfatiza a importância de unir:
- produtividade agrícola,
- responsabilidade ambiental,
- uso inteligente de recursos naturais,
- e tecnologias sustentáveis.

---

## ✨ Funcionalidades

- Layout moderno e responsivo
- Hero section com imagem de fundo impactante
- Cards informativos
- Sessão de impactos positivos
- Navegação simples
- Design adaptável para dispositivos móveis

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Responsividade com Media Queries
